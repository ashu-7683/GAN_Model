# Machine Learning Projects Portfolio

This repository contains various machine learning projects demonstrating different techniques and algorithms.

## Projects

### 1. Text-to-Image GAN
- **Location**: `text-to-image-gan/`
- **Description**: Generative Adversarial Network for generating images from text descriptions
- **Tech Stack**: TensorFlow, Keras, CNN, LSTM

### 2. Linear Regression
- **Location**: `linear-regression/`
- **Description**: Simple linear regression model to predict medical charges based on BMI
- **Tech Stack**: Scikit-learn, Pandas, Matplotlib
