# Text-to-Image GAN

A Generative Adversarial Network that generates images from text descriptions using TensorFlow and Keras.

## Model Architecture
- **Generator**: CNN-based upsampling network
- **Discriminator**: CNN-based classification network
- **Text Processing**: Tokenization and embedding

## Features
- Text-to-image synthesis
- Customizable architecture
- Training visualization

## Usage
```bash
pip install -r requirements.txt
python gan.py
